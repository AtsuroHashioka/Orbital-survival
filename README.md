# Orbital Survival

宇宙空間を舞台にした2Dサバイバルゲームです。

## 概要

プレイヤーは天体を操作し、公転軌道上を生き残ることを目指します。

## 要件

- Python 3.8 以上
- Pygame

## インストールと実行方法

1. **必要なライブラリをインストールします:**

    ```bash
    pip install pygame
    ```

2. **ゲームを実行します:**

    ```bash
    python main.py 
    ```

## 操作方法

- **[>]**: 右に移動
- **[<]**: 左に移動
